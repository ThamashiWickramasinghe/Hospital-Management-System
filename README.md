The **City Nursing Home Hospital Management System** is a Java-based application developed to simplify and automate hospital operations such as patient management, consultant management, staff handling, scheduling, and report generation. The system provides role-based access for Cashiers and Managers to ensure secure and efficient management.

## Features
- Secure login for Cashier and Manager roles
- Add, view, search, and delete patient records
- Add, view, search, and delete consultant records
- View doctor schedules and available appointments
- Generate and view hospital reports
- Manage staff details (Cashiers and Managers)
- User-friendly menu-driven interface

## User Roles
### Cashier
- Manage patients
- Manage consultants
- View reports
- View doctor schedules

### Manager
- All Cashier functionalities
- Add Cashiers
- Add Managers
- View staff details

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- File Handling / Data Storage
- Console-based UI
